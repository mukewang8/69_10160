<h3>课程目标：</h3>
<p>这是一套关于搭建渗透环境和信息收集的课程，这门课，将带你从一个初学者开始，从一个没有<a title="查看与 渗透测试 相关的文章" target="_blank">渗透测试</a>的起点开始，掌握渗透测试环境搭建，和如何搜集信息。</p>
<p>本课程专注于实际操作，包括每次实操背后的理论。在进入信息搜集之前，我们会首先学习如何搭建一套渗透测试环境，把所需的软件都安装上，以便在你自己的机器上实践信息搜集。</p>
